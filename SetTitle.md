## SetTitle ##
```
fpdf.set_title(title, isUTF8=False)
```
### Description ###

Defines the title of the document.

### Parameters ###

title:
> The title.
isUTF8:
> Indicates if the string is encoded in ISO-8859-1 (false) or UTF-8 (true).
> Default value: false.

SetAuthor, SetCreator, SetKeywords, SetSubject.
