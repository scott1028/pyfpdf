## SetAutor ##
```
fpdf.set_autor(autor, isUTF8=False)
```
### Description ###

Defines the author of the document.

### Parameters ###

author:
> The name of the author.
isUTF8:
> Indicates if the string is encoded in ISO-8859-1 (false) or UTF-8 (true).
> Default value: false.

### See also ###

SetTitle, SetCreator, SetKeywords, SetSubject.
